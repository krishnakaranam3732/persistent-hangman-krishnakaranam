Installing

Install and update using pip:

> pip install <library_name>

Required Libraries:

1. Flask
2. Flask-Cors
3. Flask-SQLAlchemy
4. psycopg2
5. SQLAlchemy
6. flask_restplus
7. gunicorn
8. PyLD

Run app.py through command line

python app.py
